Part 1#

Part one is fairly easy as all you really have to do is find the most common bit and least common bit for each bit in the string, last you convert the most common strong and the least common string to base 10 and multiply them base 10 conversion in julia is very simple as all you have to do is this:

parse(Int, value; base = 2)

the implementations for most common and least common bit are as as follows

function most_common(array::Matrix{Int}, col::Int)::Int
	rows = size(arr, 1)
	num_ones = sum(arr[:, col])
	num_zeros = rows - num_ones
	return num_ones >= num_zeros ? 1 : 0
end

and the least common bit take the xor of the most common bit string and 1.

least_common(arr::Matrix{Int}, col::Int)::Int
	= (xor(most_common(arr, col), 1))
